# Modern Portfolio Website

Next.js + Tailwind CSS + Framer Motion single-page portfolio.

## Quick Start

1. Install Node.js 18+.
2. Run:

```bash
npm install
npm run dev
```

3. Open `http://localhost:3000`.

## Customize Content

Update `src/data/portfolio.json` to change profile data, experience, projects, skills, and portfolio items.

## Features

- One-page modern layout with glassmorphism cards
- Dark/Light mode toggle
- Framer Motion scroll reveal and hover animations
- Floating hero gradient blob
- Scroll progress bar
- Particle background canvas
- SEO metadata
- Loading screen
- Blog entry page
